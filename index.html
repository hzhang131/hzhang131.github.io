Contextual Search on Wikipedia Graph
  1. Searching the wikipedia with contextual information (2008)
    2. Notes

    Key Idea: This paper improves wikipedia search by passing a context page alongside user query terms to help pinpoint the context at the wikipedia hyperlink structure where the user queries might appear.

Query Expansion / Reformulation with Wikipedia
  1. A New Approach for Query Expansion using Wikipedia and WordNet (2019)

    2. Key Idea: This paper combines two knowledge sources, Wikipedia and WordNet, to help users to reformulate user queries by taking advantage of the wikipedia hyperlink structure and the synonym/hypernym structure from WordNet

  2. Query Expansion with Freebase (2015)

    3. Key Idea: This paper presents a framework to expand user query with Freebase by first using an unsupervised algorithm to link queries to Freebase objects and select expansion terms from said objects. Then, use a supervised model to combine multiple expansion methods.

  3. Retrieval and Feedback Models for Blog Feed Search (2008)

    4. Key Idea: First part of this paper presents a wikipedia-link feedback based query expansion model where anchor phrases are extracted as expansion candidates from two sets of top ranked anchortext-linked wikipedia article sets.

  4. Understanding Graph Structure of Wikipedia (2015)

    5. Key Idea: This paper discovers that the structure of dense cycles of with a minimum amount of categories tend to identify the most relevant information.

Wikipedia Article Recommendation System
  1. Kshitij: A Search and Page Recommendation System for Wikipedia (2007)

    2. Key Idea: This paper devises a system that generates related articles of the wikipedia search results as well as the related article of the article that the user is viewing by taking advantage of the structured wikipedia data.

Searching Knowledge Databases with Human Queries
  1. SWiPE: Searching Wikipedia by Example (2012)

    2. Key Idea: This paper demonstrates a new interface that helps user to query the wikipedia database without knowledge of database query languages

Entity Linking
  1. Named entity disambiguation by leveraging wikipedia semantic knowledge （2009）

    2. Key Idea: This paper uses the wikipedia semantic network for the social relatedness between different wikipedia entities to disambiguate different entities with the same surface form text. (Anchor Text)

  2. Robust Named Entity Disambiguation with Random Walks (2016）

    3. Key Idea: This paper treats Named Entity Disambiguation as an optimization problem and presents two novel approaches to Named Entity Disambiguation with knowledge bases. Iterative and greedy approximation as well as learning to rank.

Question Answering
  1. Learning to retrieve reasoning paths over wikipedia graphs for question answering (2019)

    2. Key Idea: This paper introduces a new graph-based recurrent retrieval approach that learns to retrieve reasoning paths using an RNN over the Wikipedia graph and extract answers from the ranked reasoning paths to answer multi-hop open-domain questions.

Concept Similarity / Semantic Relatedness
  1. An Effective, Low-Cost Measure of Semantic Relatedness Obtained from Wikipedia Links (2008)

    2. Key Idea: This paper describes a new technique for obtaining measures of semantic relatedness by using the hyperlink structure of wikipedia instead of the category hierarchy or textual content.
