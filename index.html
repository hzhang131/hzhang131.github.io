Papers on Wikipedia / Knowledge Graph Search

Contextual Search on Wikipedia Graph
  1. Searching the wikipedia with contextual information (2008)

    2. Key Idea: This paper improves wikipedia search by passing a context page alongside user query terms to help pinpoint the context at the wikipedia heterogeneous graph where the user queries might appear.

Query Expansion / Reformulation with Wikipedia
  1. A New Approach for Query Expansion using Wikipedia and WordNet (2019)

    2. Key Idea: This paper combines two knowledge sources, Wikipedia and WordNet, to help users to reformulate user queries by taking advantage of the wikipedia hyperlink structure and the synonym/hypernym structure from WordNet

Wikipedia Article Recommendation System
  1. Kshitij: A Search and Page Recommendation System for Wikipedia (2007)

    2. Key Idea: This paper devised a system that generates related articles of the wikipedia search results as well as the related article of the article that the user is viewing by taking advantage of the structured wikipedia data.

Searching Knowledge Databases with Human Queries
  1. SWiPE: Searching Wikipedia by Example (2012)

    2. Key Idea: This paper demonstrates a new interface that helps user to query the wikipedia database without knowledge of database query languages
